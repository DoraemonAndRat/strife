Strife

2D Game Library

Strife is a 2d game library for making game development in Go easier.

Setting up is as simple as cloning the repository and building it with the go compiler.

To use Strife, import the package into a new go project.

clone the repository

go get github.com/felixangell/strife

cd into the repo

cd $GOPATH/src/github.com/felixangell/strife

cd to an example

cd example/particles

compile & run the example

go run particles.go
